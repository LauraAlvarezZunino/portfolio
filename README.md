Portfolio Personal – Single Page Application (SPA)
==================================================

Portfolio personal desarrollado como Single Page Application (SPA) utilizando JavaScript puro, con diseño limpio, responsive y orientado a la experiencia del usuario.Incluye una sección sobre mí, un listado de proyectos y un formulario de contacto integrado mediante EmailJS.

Tecnologías utilizadas
----------------------

*   HTML5
    
*   CSS3
    
*   JavaScript Vanilla
    
*   EmailJS
    

Características principales
---------------------------

### Single Page Application (SPA)

Navegación interna sin recargar la página mediante manipulación del DOM y render dinámico de contenido.

### Sección Sobre mí

Información personal, habilidades y breve presentación profesional.

### Sección de Proyectos

Listado de proyectos con descripción, tecnologías utilizadas y enlaces a repositorios o demos.

### Formulario de Contacto con EmailJS

Formulario funcional sin necesidad de backend propio.Incluye validaciones, manejo de estados y confirmación de envío.

### Diseño Responsive

Adaptado para móviles, tablets y escritorios utilizando prácticas modernas de CSS.
